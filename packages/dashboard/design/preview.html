Anti-slop confirmed

Avoided
  • Purple/violet gradients (gold solid only)
  • 3-column icon grid (numbered list sidebar)
  • Centered everything (left-anchored, asymmetric)
  • Decorative blobs (hairline borders only)
  • Gradient CTAs (solid accent fill)
  • Generic stock-photo hero (no images at all)
Deliberate risks vs category
  • Serif body instead of sans — spec body is read 80% of session, serif aids long-form prose
  • Antique gold accent instead of indigo/purple — zero dev tools use gold, instant brand
  • Sidebar as numbered manifest, not icon strip — Fraunces italic for phase numbers
  • ID popovers borderless with 1px top gold rule — instrument readout aesthetic
